# 🌸 K-Nearest Neighbors (KNN) Classifier - Iris Dataset

This project demonstrates the implementation of the **K-Nearest Neighbors (KNN)** algorithm using the famous **Iris dataset**. It includes data preprocessing, model training, prediction, and evaluation.

---

## 📌 Project Overview

- Algorithm: K-Nearest Neighbors (KNN)
- Dataset: Iris Dataset (from sklearn)
- Objective: Classify iris flowers into different species based on features

---

## ⚙️ Technologies Used

- Python 🐍
- Scikit-learn
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

## 📊 Steps Performed

1. Loaded the Iris dataset
2. Split data into training and testing sets
3. Applied feature scaling using StandardScaler
4. Trained KNN model (k = 5)
5. Made predictions on test data
6. Evaluated model using:
   - Accuracy Score
   - Classification Report
